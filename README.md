# Calculator
A simple calculator made with Flask and JavaScript.

## Versions
Flask==0.12.2 
Jinja2==2.8  
Werkzeug==0.11.11  

## Installation
Change to app directory, use `virtualenv` create and activate virtual enviroment.  
Then use `pip` to install requirementsï¼š  
```
pip install flask
```
Run:  
```
set FLASK_APP=app.py    
flask run
```
